# Контент материалов

Здесь хранятся конспекты тем в формате Markdown.

## Как добавить тему

1. Напишите тему в Obsidian и экспортируйте `.md`-файл.
2. Положите файл в нужную папку:
   - физика → `content/materials/physics/<slug>.md`
   - математика → `content/materials/math/<slug>.md`
3. Имя файла (`<slug>`) должно совпадать со slug темы из
   `lib/content/materials.ts` (например, `ohm-law.md`).
4. Готово – после пересборки/деплоя тема откроется на сайте.

Если для темы файла ещё нет, на странице показывается «Материал готовится».

## Что поддерживается в Markdown

- Заголовки `##`, списки, **жирный**/_курсив_, ссылки, изображения.
- Таблицы (GitHub-разметка).
- Формулы LaTeX: `$...$` в строке и `$$...$$` отдельным блоком (рендерит KaTeX).
- Цитаты `>` и блоки кода.

Примеры готовых тем: `physics/ohm-law.md`, `math/progressions.md`.
