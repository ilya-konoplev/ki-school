# 5. Как добавить лабораторную работу

Виртуальная лабораторная — это маленькая интерактивная страничка (HTML/CSS/JS),
которая открывается прямо на сайте во встроенном окне. Лабы живут в разделе
**Материалы → Физика → Виртуальные лабораторные**.

Добавление состоит из двух частей: положить файлы лабы и добавить её в список.

---

## Что такое «лаба» технически

Это папка с файлами, у которой есть точка входа — файл `index.html`. Например,
готовый пример лежит здесь: `public/labs/pendulum/index.html` (модель маятника).

---

## Шаг 1. Положить файлы лабы

1. Подготовь папку с лабой. Внутри обязательно должен быть `index.html`
   (остальные файлы — стили, скрипты, картинки — рядом). Дай папке короткое имя
   латиницей, например `lenses`.
2. На [github.com](https://github.com) открой репозиторий `ki-school` → папку
   `public` → `labs`.
3. **Add file → Upload files** и перетащи свою папку с лабой целиком (GitHub
   сохранит структуру). Внизу — **Commit changes**.

Должно получиться `public/labs/lenses/index.html`.

---

## Шаг 2. Добавить лабу в список

Список лаб задаётся в файле `lib/content/labs.ts`. Добавим туда запись.

1. На GitHub открой файл `lib/content/labs.ts`, нажми карандаш ✎ (Edit).
2. Внутри списка `labs` добавь новый блок по образцу (не забудь запятую между
   записями):

   ```ts
   {
     slug: "lenses",
     title: "Линзы и построение изображений",
     description: "Двигайте предмет и линзу — стройте изображение.",
     src: "/labs/lenses/index.html",
     topicSlug: "lenses",
   },
   ```

   Пояснения:
   - `slug` — короткое имя латиницей (можно как имя папки).
   - `title` — название, которое увидят на сайте.
   - `description` — короткое описание.
   - `src` — путь к файлу: `/labs/ИМЯ-ПАПКИ/index.html`.
   - `topicSlug` — необязательно; привязка к теме физики (slug темы из админки →
     Материалы).

3. Внизу — **Commit changes**.

Через ~1 минуту лаба появится в списке на странице
**Материалы → Физика → Виртуальные лабораторные**.

---

## Советы

- Лаба должна работать «сама по себе»: всё нужное — внутри её папки, без ссылок
  на внешние файлы с компьютера.
- Если лаба не открылась — проверь, что файл называется именно `index.html` и что
  путь в `src` совпадает с папкой (`/labs/ИМЯ/index.html`).
- Открой пример `public/labs/pendulum/index.html` как образец того, как может
  быть устроена простая лаба.
- Не уверен в редактировании `labs.ts`? Пришли мне готовую папку с лабой —
  добавлю в список сам.
