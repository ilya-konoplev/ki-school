# 3. Как добавить тему (материал)

Темы по физике и математике — это страницы в разделе **Материалы**. Список всех
тем уже создан (по кодификатору ОГЭ). Пока у темы нет текста, на её странице
написано «Материал готовится». Твоя задача — написать текст и положить его файлом.

Каждая тема — это один файл в формате **Markdown** (`.md`). Писать удобно в
**Obsidian**, а потом добавить файл на сайт.

---

## Шаг 1. Узнай имя файла для темы

У каждой темы есть строго заданное имя файла. Посмотреть его проще всего в
админке:

1. Войди в админку → меню **Материалы**.
2. Найди нужную тему — справа от неё показано имя файла, например:
   `physics/ohm-law.md` или `math/progressions.md`.

Первая часть (`physics` или `math`) — это папка (предмет), вторая
(`ohm-law.md`) — имя файла.

---

## Шаг 2. Напиши тему в Obsidian

1. Создай в Obsidian новую заметку и напиши материал.
2. Используй обычную разметку Markdown: заголовки, списки, **жирный** текст,
   формулы, картинки, таблицы. Шпаргалка — в разделе [4. Markdown](04-markdown.md).
3. Сохрани/экспортируй заметку как файл `.md`. Назови файл точно так, как
   показано в админке (например, `ohm-law.md`).

> Файл из Obsidian обычно уже `.md` и лежит в папке хранилища (vault). Можно
> просто взять его оттуда.

---

## Шаг 3. Добавь файл на сайт

Самый простой способ — прямо на сайте GitHub, без программ:

1. Зайди на [github.com](https://github.com), открой свой репозиторий
   `ki-school`.
2. Перейди в папку нужного предмета:
   - физика: `content` → `materials` → `physics`
   - математика: `content` → `materials` → `math`
3. Нажми **Add file → Upload files**.
4. Перетащи свой `.md`-файл в окно. Убедись, что имя файла правильное
   (как в админке).
5. Внизу нажми **Commit changes**.

Через ~1 минуту Vercel обновит сайт, и тема откроется (вместо «Материал
готовится» появится твой текст).

> Альтернатива через **GitHub Desktop**: положи файл в папку
> `content/materials/physics` (или `math`) на компьютере → в GitHub Desktop
> увидишь изменение → **Commit** → **Push**.

---

## Как обновить или удалить тему

- **Обновить:** загрузи файл с тем же именем заново (GitHub предложит заменить) —
  или открой файл на GitHub, нажми карандаш ✎, поправь и **Commit**.
- **Удалить текст:** удали файл темы на GitHub (открыть файл → значок корзины →
  Commit). Тема снова покажет «Материал готовится».

---

## Частые вопросы

- **Можно ли назвать файл по-русски?** Нет. Имя файла должно точно совпадать с
  тем, что показано в админке (латиницей), иначе сайт не «увидит» тему.
- **Тема не появилась.** Проверь: файл лежит в правильной папке
  (`physics`/`math`), имя точно как в админке, и прошла минута после Commit.
- **Нужно добавить совсем новую тему, которой нет в списке?** Список тем задан в
  коде. Напиши мне (разработчику) — добавлю тему в список, дальше наполнишь
  файлом как обычно.
