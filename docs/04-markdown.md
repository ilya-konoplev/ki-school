# 4. Шпаргалка по Markdown

Markdown — это простой способ оформлять текст: ты пишешь обычными символами, а
сайт превращает их в красивые заголовки, списки, формулы и таблицы. Ниже —
только то, что реально пригодится для тем.

Слева — что писать, справа — что получится.

---

## Заголовки

```markdown
## Большой заголовок раздела
### Заголовок поменьше
```

Не используй `#` (один) — он зарезервирован под название темы. Начинай с `##`.

## Выделение текста

```markdown
**жирный**, *курсив*
```

## Списки

Маркированный (точки):

```markdown
- первый пункт
- второй пункт
```

Нумерованный:

```markdown
1. шаг один
2. шаг два
```

## Формулы (физика и математика)

Формулы пишутся на языке LaTeX. Есть два варианта.

**В строке** — между одинарными `$`:

```markdown
Скорость считается как $v = s / t$.
```

**Отдельной строкой по центру** — между `$$`, причём `$$` должны стоять на
**отдельных строках**:

```markdown
$$
I = \frac{U}{R}
$$
```

> Важно: если написать `$$ ... $$` в одну строку, формула получится маленькой
> (как в тексте). Для крупной формулы по центру — `$$` на отдельных строках.

Полезные обозначения LaTeX:

| Что нужно        | Как писать            | Получится         |
| ---------------- | --------------------- | ----------------- |
| Дробь            | `\frac{a}{b}`         | a/b в виде дроби  |
| Степень          | `x^2`                 | x²                |
| Индекс           | `a_1`                 | a₁                |
| Корень           | `\sqrt{x}`            | √x                |
| Умножение (точка)| `a \cdot b`           | a·b               |
| Греческие буквы  | `\alpha`, `\pi`       | α, π              |
| Не равно         | `\neq`                | ≠                 |

## Таблицы

```markdown
| Величина | Обозначение | Единица |
| -------- | ----------- | ------- |
| Сила     | F           | Н       |
| Масса    | m           | кг      |
```

Первая строка — заголовки, вторая (с дефисами) — обязательный разделитель.

## Цитата / важная мысль

```markdown
> Это выделенная заметка или важное правило.
```

## Ссылка

```markdown
[текст ссылки](https://example.com)
```

## Картинки

1. Положи картинку в папку `public/images/` в репозитории (через GitHub:
   **Add file → Upload files**), например `public/images/lens.png`.
2. В тексте темы вставь:

   ```markdown
   ![Схема линзы](/images/lens.png)
   ```

   В квадратных скобках — подпись (если картинка не загрузится), в круглых —
   путь, начиная со `/images/`.
