# ki-school

Личный сайт репетитора по физике и математике (подготовка к ОГЭ).
_konoplev ilya — school_

## Стек

- **Next.js 15** (App Router) + TypeScript
- **Tailwind CSS v4** — дизайн-система на CSS-переменных
- **Supabase** — база данных, файлы, аутентификация (подключается в п.4)
- **Vercel** — хостинг (регион Frankfurt `fra1`)

## Запуск локально

```bash
npm install
cp .env.local.example .env.local   # затем впишите ключи Supabase
npm run dev
```

Открыть http://localhost:3000

## Скрипты

| Команда         | Назначение                       |
| --------------- | -------------------------------- |
| `npm run dev`   | Локальная разработка (Turbopack) |
| `npm run build` | Продакшн-сборка                  |
| `npm start`     | Запуск собранного приложения     |
| `npm run lint`  | Проверка ESLint                  |

## Дизайн-система

- Цветовые токены и темы — в [`app/globals.css`](app/globals.css).
  Светлая тема по умолчанию; тёмная включается классом `.dark` на `<html>`
  (переключатель в шапке, выбор сохраняется в `localStorage`).
- Фирменный акцент: `#9974f7`.
- Шрифты: **Inter** (текст/интерфейс, кириллица), **Fraunces** (акцентный
  serif для латиницы), **JetBrains Mono** (код/формулы) — см.
  [`app/layout.tsx`](app/layout.tsx).
- Логотип (оси x/y + график ∛x) — [`components/brand/Logo.tsx`](components/brand/Logo.tsx).

## Гайд для Ильи (без программирования)

Пошаговые инструкции — в папке [`docs/`](docs/README.md):
запуск и деплой, работа с админкой, добавление тем и лабораторных, шпаргалка по
Markdown.

## Структура

```
app/                 страницы (App Router)
components/
  brand/             логотип
  theme/             провайдер и переключатель темы
lib/
  supabase/          клиент Supabase (browser/server)
```
